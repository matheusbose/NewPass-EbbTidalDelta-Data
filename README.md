# NewPass-EbbTidalDelta-Data
Data and analysis files supporting the manuscript "Natural and Anthropogenic Controls on the Decadal Evolution of an Ebb-Tidal Delta on a Low-Energy, Microtidal Coast: New Pass, Florida (USA).

Folders
## Polar_Analysis
Contains the scripts and input data used to perform the polar analysis.

## Volume
Contains the scripts and input data used for the volumetric analysis.

### Chronostratigraphy
Contains the scripts and input data used for the chronostratigraphic analysis.
